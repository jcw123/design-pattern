# 求连续范围的数据某一个质因数出现的总数模板


## 模板
```java
// 其中n表示1 ~ n这一个连续的序列，p表示其中的一个质因数
int count(int n, int p) {
    int cnt = 0;
    while(n != 0) {
        cnt += n / p;
        n /= p;
    }
    return cnt;
}
```

## 思考
5 5 * 2 5 * 3 5 * 5  5* 5  5 * x  => 用 n /5 可以获得x值，即第一次出现5的总数，在这个基础上，再 计算1 2 3 ... x 中第一次出现5的总数
