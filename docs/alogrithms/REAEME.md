# 算法

## 核心思想
1、分块思想
可以利用临界值进行分块，小于临界值用一种解法，大于临界值用另一种解决，也可以降低算法的时间复杂度，有的题如果暴力解法可以解出来，接下来就看看是否能够用一些高效的算法或者数据结构降低空间复杂度或者时间复杂度；最常用的分块思想还是二分法，无敌呀，最主要是如果想出二分
