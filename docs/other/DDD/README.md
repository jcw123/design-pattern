## 备注
1、 避免职责过载
对象承载的职责应该是清晰且明确的，如果承载的职责过多，会导致模型过于复杂，难于理解；

2、进行不同层次的接口设计时，需要显式揭露接口的意图及模型的细节，防止去使用时去感知过多的细节；
