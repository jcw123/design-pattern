## 可转债

### 指标相关
- 转股价值
转股价值 = 可转债面值 / 转股价格 * 正股价格

## 思考
1、当投资涨了之后，只有落袋为安才算赚了，如果没有转换为真金白银，就没法算最终盈利了。

2、买新股时，前面四天坚决不买，等第五天看看形式，看是否需要买。
