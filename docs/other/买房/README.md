# 买房攻略

## 看房标准
1、房子方正，面积能够充分利用
2、需要房子满二或者满五唯一，不然需要缴纳多余的税，尤其没有满二的情况下，需要多交几十万
3、优先买新房，没有新房的话，买二手房，但是需要是次新；
4、看好户型，看好一层居住的人数，邻居也非常重要；
