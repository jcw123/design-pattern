# 二进制位运算函数

## 尾部0的个数
```c++
__builtin_ctz(64)
__builtin_ctzll(64)
```

## 前导0的个数
```cpp
__builtin_clz(64) 
__builtin_clzll(64)
```

## 二进制表示中1的个数
```cpp
__builtin_popcount(64)
```

## 二进制表示中1的个数奇偶性
```cpp
__builtin_parity(64)
```

## 二进制表示中最后一个1在第几位
```cpp
__builtin_ffs(64)
```

## 快速开平方
```cpp
__built_sqrt(64)
__built_sqrtf(64)
```
