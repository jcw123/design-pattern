# yaml

## 数组
```yaml
i_am_list: # 构建一个列表
  - v1
  - v2
  - v3
  - v4 
```

## map
```yaml
i_am_map: # 构建一个map
  key1: v1
  key2: v2
  key3:
    t1: v3
    t2: v4
```
